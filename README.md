# Beauty Store 

## Descrição do Projeto

Beauty Store é uma aplicação web desenvolvida para gerenciamento e visualização de produtos cosméticos. O sistema permite que usuários visualizem produtos cadastrados e que administradores realizem operações completas de cadastro, edição e exclusão por meio de um painel administrativo moderno.

O projeto foi desenvolvido utilizando apenas HTML, CSS e JavaScript puro (Vanilla JavaScript), consumindo uma API REST hospedada no Render.

---

# Objetivo

Desenvolver uma aplicação frontend responsiva capaz de consumir uma API REST real, aplicando conceitos de manipulação do DOM, requisições assíncronas, autenticação e operações CRUD.

## Área Administrativa

---

# Operações CRUD

## GET

Busca todos os produtos cadastrados.

## POST

Realiza o cadastro de novos produtos.

## PATCH

Atualiza informações de produtos existentes.

## DELETE

Remove produtos cadastrados.

# Decisões Técnicas

* Utilização de JavaScript modular para organização do código.
* Manipulação do DOM sem uso de frameworks.
* Separação das páginas em módulos independentes.
* Consumo da API utilizando Fetch API.
* Interface responsiva desenvolvida com CSS puro.
* Armazenamento do token de autenticação no LocalStorage.
