

## Descrição do Projeto

Beauty Store é uma aplicação web desenvolvida para gerenciamento e visualização de produtos cosméticos. O sistema permite que usuários visualizem produtos cadastrados e que administradores realizem operações completas de cadastro, edição e exclusão por meio de um painel administrativo.

O projeto foi desenvolvido utilizando apenas HTML, CSS e JavaScript puro (Vanilla JavaScript), consumindo uma API REST hospedada no Render.


##  Objetivo

Desenvolver uma aplicação frontend responsiva capaz de consumir uma API REST real, aplicando conceitos de:

- Manipulação do DOM
- Requisições assíncronas
- Autenticação com token
- Operações CRUD
- Organização de código em JavaScript puro

##  Funcionalidades Implementadas

###  Área Pública

- Visualização dos produtos cadastrados
- Banner principal da loja
- Layout responsivo
- Navegação entre páginas

###  Área Administrativa

- Login de administrador
- Cadastro de produtos
- Edição de produtos
- Exclusão de produtos
- Listagem dinâmica de produtos
- Dashboard administrativo
- Consumo de API em tempo real

---

##  Operações CRUD

### GET
Lista todos os produtos cadastrados.

### POST
Cria novos produtos.

### PATCH
Atualiza produtos existentes.

### DELETE
Remove produtos cadastrados.

## Decisões Técnicas

- Uso de JavaScript modular para organização do projeto
- Manipulação do DOM sem frameworks
- Separação em módulos (admin, login, app)
- Consumo da API com Fetch API
- Interface responsiva com CSS puro
- Autenticação com token via LocalStorage
- Tratamento de imagem padrão para evitar quebra de layout
